# Regression Algorithms Implementation (Mini ML Projects)

This repository contains implementations of fundamental **Regression algorithms** built as part of a machine learning learning portfolio. The projects focus on understanding the mathematical intuition and practical implementation of regression techniques used for predictive modeling.

The notebooks demonstrate how regression algorithms work from scratch and using common machine learning approaches.

---

## 📌 Repository Contents

This repository currently contains the following Jupyter Notebooks:

### 1. Linear Regression Implementations

This notebook explores multiple approaches to implementing Linear Regression, including optimization techniques and polynomial extensions.

Algorithms covered:

* Linear Regression using **Gradient Descent**
* Linear Regression using **Normal Equation**
* **Polynomial Regression** with different polynomial degrees

Key concepts demonstrated:

* Cost Function (Mean Squared Error)
* Gradient Descent optimization
* Effect of polynomial degree on model complexity
* Underfitting vs Overfitting

---

### 2. Ridge Regression

This notebook implements **Ridge Regression**, a regularized version of linear regression designed to reduce overfitting by penalizing large coefficients.

Key concepts demonstrated:

* Regularization in regression
* L2 penalty
* Effect of regularization strength (λ)
* Comparison with standard Linear Regression
